## How to run the project locally

1. `git clone https://github.com/yonicelleyva/fe-exercise.git`
2. `cd fe-exercise`
3. `npm install`
4. `npm start` to run project

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
